# CV | Bhuwan Prasad Upadhyay

Web: <https://bhuwanupadhyay.github.io/cv/>

## How to use ??
- Fork this repo.
- Install dependencies.
```shell script
make install
```
- Change cv detail in data file `_data/cv.yml`.
- If you want to change colors modify `assets/css/theme.scss`.
```scss
...

$primary: red;
$secondary: blue;

...
```
- Run
```
make run
```
